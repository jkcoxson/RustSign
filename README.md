# RustSign
A Rust library for signing iOS apps

# About
This library is meant to bridge the gap that AltSign creates, and make it bindable to "C" FFI compliant languages.
The goal of this library is to translate all of AltSign into Rust, but until then there are C++ abstractions in place.
Until this is completely translated to Rust, this library will only work on Linux.
